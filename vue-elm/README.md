# 技术栈

> vue + vuex + vue-router + element-ui + sass + axios + es6/7

## 项目运行
git clone https://github.com/123456yr/elm-yr.git
cd elm-yr/vue-elm
npm start

## 关于接口

接口数据大部分来自饿了么官网的数据，
详情页中由于总是获取不到官网的数据，将官网中某一个商品的详情信息复制到static/json中，
还有 搜索 获取商品时总是失败，就用了首页的数据
